# End-To-End-data-analytics-python-sql-powerbi
An end-to-end data analytics project covering data cleaning, Python, EDA, SQL analysis, and Power BI dashboarding to uncover customer behavior insights and support data-driven decisions.

# 📊 End-to-End Data Analytics Project

## 📌 Overview

This project demonstrates a complete **data analytics workflow**, starting from data loading and cleaning to visualization and reporting.
It highlights the use of **Python, SQL, and Power BI** to extract meaningful insights and present them in a clear and actionable format.

---

## 📂 Dataset

The dataset used in this project contains structured data related to customer behavior and transactions.

**Key fields include:**

* Customer ID
* Demographics (Age, Gender, Location)
* Purchase details
* Product category
* Subscription status

---

## 🛠️ Tools & Technologies

* **Python** (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning & EDA
* **SQL (PostgreSQL / MySQL / SQL Server)** – Data querying & analysis
* **Power BI** – Dashboard creation & visualization
* **Gamma** – Presentation (PPT) creation
* **Excel / CSV** – Data storage

---

## 🔄 Project Steps

### 1. Data Loading

* Imported dataset using Python (Pandas)
* Verified structure, data types, and missing values

### 2. Data Cleaning

* Handled missing values
* Removed duplicates
* Standardized column formats

### 3. Exploratory Data Analysis (EDA)

* Analyzed distributions and trends
* Identified patterns in customer behavior
* Created visualizations using Python

### 4. SQL Analysis

* Loaded cleaned data into database
* Performed queries such as:

  * Aggregations (SUM, AVG, COUNT)
  * Grouping and filtering
  * Customer-level insights

### 5. Dashboard Development

* Built an interactive **Power BI dashboard**
* Added KPIs, charts, and slicers for dynamic analysis

### 6. Reporting

* Summarized insights in a structured report
* Created a presentation using **Gamma**

---

## 📊 Dashboard

The Power BI dashboard includes:

* Key performance indicators (KPIs)

* Customer segmentation

## 🚀 How to Run

### 🔹 Python (EDA & Cleaning)

1. Install required libraries:

   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
2. Run the Python script or Jupyter Notebook

---

### 🔹 SQL

1. Load dataset into your database (PostgreSQL/MySQL/SQL Server)
2. Run the SQL queries provided in the project

---

### 🔹 Power BI

1. Open the `.pbix` file in Power BI Desktop
2. Refresh data if needed
3. Interact with dashboard visuals

---

### 🔹 Presentation

* Open the generated PPT (Gamma export) for project summary
